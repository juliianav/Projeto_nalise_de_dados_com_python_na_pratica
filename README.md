# Análise de Dados - Materiais de Agricultura

## 📌 Sobre o Projeto

Análise exploratória de dados (EDA) sobre materiais de agricultura, desenvolvida em Python utilizando o Google Colab. O projeto foca em limpeza, organização e visualização dos dados para geração de insights.

---

## 🛠️ Tecnologias

* Python
* Pandas
* Matplotlib
* Google Colab

---

## 📊 Etapas

* Importação e leitura dos dados
* Limpeza e tratamento
* Análise exploratória (EDA)
* Visualização de dados

---

## 📈 Insights

* Identificação de padrões nos dados
* Tratamento de inconsistências
* Geração de informações úteis para análise

---

## ▶️ Como Executar

1. Abrir o arquivo `.ipynb` no Google Colab
2. Executar as células em sequência
